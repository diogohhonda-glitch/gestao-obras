🔌 Integrações

🏦 Fase 2 — Asaas (Bancário)
Boletos, PIX, cobranças e conciliação bancária automática
Passo a passo:
1. Acesse asaas.com → crie conta gratuita com CNPJ 31.527.551/0001-31
2. Conclua o cadastro e verificação da empresa
3. Vá em Configurações → Integrações → API
4. Copie sua API Key
5. Cole abaixo para ativar emissão de boletos e PIX direto do sistema
⚠️ Conta gratuita: sem mensalidade. Cobranças: ~1-3% por boleto pago.
🏛️ Fase 3 — eNotas (NF-e/NFS-e)
Emissão e recepção de notas fiscais integrado à Receita Federal
Passo a passo:
1. Adquira um certificado digital A1 e-CNPJ (~R$250/ano em qualquer certificadora)
2. Acesse enotas.com.br → crie conta → cadastre seu CNPJ 31.527.551/0001-31
3. Faça upload do certificado A1 (.pfx) e informe a senha
4. Configure o município (prefeitura) para NFS-e de serviços
5. Vá em API → Gerar token → copie e cole abaixo
⚠️ Custo: ~R$79/mês (até 50 NFs). Certificado digital: ~R$250/ano.
